# ModMenuCrew - Among Us Mod Menu

A mod menu for Among Us with cheats and utilities. No license key required.

## Features

- **No Kill Cooldown** - Instant kills as impostor (toggle with F9)
- **Force Votes** - Force all players to vote for a target (host only)
- **Kick/Ban Players** - Kick or ban players during a match (host only)
- **Teleport** - Teleport to other players
- **Speed Hack** - Adjust movement speed
- **No Clip** - Walk through walls
- **ESP/Tracers** - See players through walls
- **Unlock All Cosmetics** - Access all skins, hats, pets
- **Radar** - Mini-map with player positions
- And more...

## Installation

1. Download or clone this repository
2. Copy ALL files to your Among Us game folder:
   ```
   C:\Program Files (x86)\Steam\steamapps\common\Among Us\
   ```
3. Replace any existing files if prompted
4. Launch Among Us

## Controls

- **Delete** or **F1** - Open/close mod menu
- **F9** - Toggle No Kill Cooldown

## Requirements

- Among Us (Steam version)
- Windows

## Notes

- Some features require you to be the host
- Use at your own risk in public lobbies
